# Uzaktan Kumanda Sistemi

Bu proje, RF/Bluetooth tabanlı bir uzaktan kumanda sistemi içermektedir.  
Verici (transmitter) ve alıcı (receiver) kodları Arduino/STM32/C++ ile yazılmıştır.

## Özellikler
- Basit ve güvenilir haberleşme
- Çoklu kanal desteği
- Hata kontrol mekanizmaları

## Donanım
- Arduino Uno / STM32 (kullandığınız mikrodenetleyici)
- NRF24L01 / HC-05 Bluetooth modülü (kullandığınız haberleşme modülü)
- Butonlar, joystick vb.

## Kurulum ve Kullanım
1. Verici klasöründeki kodu mikrodenetleyiciye yükleyin.  
2. Alıcı cihazda alıcı kodunu çalıştırın.  
3. Donanımı şemaya göre bağlayın.
